  Le lien entre notification et géométrie du débat

  La notification de progression ("80% de consensus atteint") semble être un simple seuil + alerte. Mais si on la regarde avec le prisme de la
  géométrie du débat, c'est beaucoup plus que ça — c'est la dimension temporelle de la géométrie.

  Reprenons les briques qu'on a posées dans le PLAN :

  ┌──────────────────────────────┬───────────────────────────────────────┬───────────────────────────┐
  │            Brique            │           Ce qu'elle montre           │         Dimension         │
  ├──────────────────────────────┼───────────────────────────────────────┼───────────────────────────┤
  │ Pastilles de marge (1.4)     │ Où sont les extractions, quel statut  │ Spatiale (dans le texte)  │
  ├──────────────────────────────┼───────────────────────────────────────┼───────────────────────────┤
  │ Dashboard de consensus (1.4) │ Compteurs et % de consensus           │ Statistique (agrégée)     │
  ├──────────────────────────────┼───────────────────────────────────────┼───────────────────────────┤
  │ Heat map (1.15)              │ Intensité du débat par passage        │ Thermique (densité)       │
  ├──────────────────────────────┼───────────────────────────────────────┼───────────────────────────┤
  │ Alignement (1.12)            │ Répartition des hypostases cross-docs │ Structurelle (couverture) │
  ├──────────────────────────────┼───────────────────────────────────────┼───────────────────────────┤
  │ Filtre contributeur (2.4)    │ Qui a dit quoi                        │ Sociale (participation)   │
  ├──────────────────────────────┼───────────────────────────────────────┼───────────────────────────┤
  │ Notification (celle-ci)      │ Comment le débat évolue dans le temps │ Temporelle (mouvement)    │
  └──────────────────────────────┴───────────────────────────────────────┴───────────────────────────┘

  Chacune est une facette du dodécaèdre. La notification ne dit pas juste "80% atteint" — elle dit "la géométrie du débat a changé de forme". Quand
   un CONTROVERSÉ devient CONSENSUEL, c'est un mouvement dans le dodécaèdre. Quand 3 nouvelles extractions apparaissent sans commentaire, c'est un
  déséquilibre. Quand un contributeur commente massivement mais que les autres sont silencieux, c'est une asymétrie.

  Ce qui est intéressant, c'est que les seuils intéressants ne sont pas que le consensus. Quelques exemples de mouvements géométriques
  significatifs :

  - "5 nouvelles extractions non commentées depuis 3 jours" → le débat stagne, zone froide
  - "Michel a commenté 12 extractions, les 3 autres contributeurs n'ont rien dit" → déséquilibre de participation
  - "Le document 'Charte v2' a 0 extraction de type PRINCIPE alors que 'Charte v1' en avait 4" → gap d'alignement (régression ?)
  - "80% de consensus atteint" → le dodécaèdre se rapproche de sa forme complète, la synthèse est possible
  - "Toutes les CONJECTURES sont DISCUTÉES mais aucun AXIOME n'a été extrait" → le débat couvre les hypothèses mais pas les fondements

  La notification est le signal que quelque chose a bougé dans la géométrie. Pas un jugement — une observation.

  Je propose de structurer ça en deux niveaux :
  - Phase 1 : notifications simples basées sur le dashboard existant (seuil de consensus, extractions orphelines)
  - Phase 2+ : notifications riches basées sur la géométrie multi-axes (participation, couverture, équilibre des hypostases) — qui se connectent
  directement au concept de Jean et Dominique

  Et au passage, c'est le bon moment pour formaliser le concept de géométrie du débat comme un objet du PLAN — pas juste une note en passant dans
  la heat map, mais une section qui relie toutes les facettes.
