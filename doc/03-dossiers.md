# Organisation par dossiers

## Creer un dossier

Cliquez sur le bouton **`+`** a droite du titre "Bibliotheque". Un champ de texte apparait : saisissez le nom du dossier et validez avec Entree.

## Classer une page dans un dossier

Les pages importees arrivent dans la section **"Non classees"** :

![Page non classee dans l'arbre](./images/07-page-non-classee.png)

Pour deplacer une page vers un dossier :

1. Cliquez sur l'icone de deplacement (fleches croisees) a cote du nom de la page.
2. Une boite de dialogue s'ouvre avec la liste de vos dossiers.
3. Selectionnez le dossier cible et cliquez sur **"Deplacer"**.

La page apparait alors sous le dossier choisi dans l'arbre, et disparait de la section "Non classees".

## Deplier / replier un dossier

Cliquez sur la ligne du dossier (nom ou fleche) pour deplier ou replier son contenu. Le nombre de pages est affiche a droite du nom.

## Retirer une page d'un dossier

Vous pouvez reclasser une page en choisissant **"— Aucun dossier —"** dans la boite de dialogue de deplacement. La page retourne dans la section "Non classees".
