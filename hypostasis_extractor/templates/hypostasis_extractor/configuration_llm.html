Comment fonctionne un analyseur ?

Un analyseur envoie un texte a une IA pour en extraire des informations.

Il se compose de deux parties :

  • Le prompt : les instructions envoyees a l'IA. Elles expliquent quoi chercher dans le texte (des personnes, des dates, des arguments, etc.).
  • Les exemples : des textes avec les bonnes reponses deja remplies. L'IA apprend de ces exemples pour mieux extraire. Plus il y a d'exemples, meilleur est le resultat.

Les options de contexte permettent d'injecter automatiquement les extractions existantes ou le texte original dans le prompt.
