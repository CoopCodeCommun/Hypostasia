{# Parcours d'onboarding en 4 etapes — affiche quand la base est vide #} {# / 4-step onboarding guide — displayed when the database is empty #}

Bienvenue dans Hypostasia

Quatre étapes pour commencer votre lecture délibérative.

{# === Etape 1 : Importer un document === #} {# / === Step 1: Import a document === #}

1. Importez votre premier document

PDF, DOCX, Markdown, fichier audio… Hypostasia extrait le texte et prépare la lecture.

{# === Etape 2 : Lancer une extraction IA === #} {# / === Step 2: Launch an AI extraction === #}

2. Lancez une extraction IA

Un analyseur IA parcourt le texte et identifie les passages clés : hypothèses, définitions, paradoxes, données…

{# === Etape 3 : Lire les hypostases === #} {# / === Step 3: Read the hypostases === #}

3. Lisez les hypostases

Ce ne sont pas des surligneurs : chaque badge colore représente un type épistémique — conjecture, axiome, phénomène… Un modèle pour comprendre comment un texte pense, pas seulement ce qu'il dit.

{# === Etape 4 : Commenter et debattre === #} {# / === Step 4: Comment and debate === #}

4. Commentez et débattez

Chaque extraction peut être commentée par plusieurs lecteurs. Le statut évolue : consensuel, discutable, discuté, controversé. Le consensus se construit collectivement.
